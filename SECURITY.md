# Security policy

Please report suspected vulnerabilities privately through GitHub's **Report a vulnerability** flow for this repository, if available, or through TeamGrid's established private security contact. Do not create a public issue.

Include the affected URL or package version, impact, reproduction steps, and any non-sensitive supporting evidence. Do not include live API credentials, webhook secrets, customer data, or personal data. Revoke any credential disclosed during testing before submitting the report.

Documentation mistakes that expose no sensitive information may be reported as ordinary issues.
